# React Slidy

This component aims to fullfil a basic need for a carousel in React. 1:1 slide, basic transition, mobile first experience and basic stuff for adding and controlling events. It's not meant to be full of options but to achieve 60fps.

**It only has one dependency**: an intersection observer polyfill that will be loaded only if your browser doesn't support it, or if you haven't polyfilled it yet.

**tl;dr: a simple and minimal slider component for React**

import { Playground, Props } from 'docz'
import ReactSlidy from './index.js'
import Number from './examples/Number'
import './index.scss'

<Playground>
  <ReactSlidy>

    <img src="/public/2.jpg" />
    <img src="/public/3.jpg" />
    <img src="/public/4.jpg" />
  </ReactSlidy>
</Playground>

If you're using sass, you could modify the next variables:

```sass
/* variables available */
$react-slidy-c-background: #dcdcdc !default;
$react-slidy-c-nav-background: rgba(255, 255, 255, .8) !default;
$react-slidy-c-nav-color: #aaaaaa !default;
$react-slidy-c-transparent: rgba(0, 0, 0, 0) !default;
$react-slidy-mh: 50px !default;
$react-slidy-h-image: auto !default;

/* then import from a sass file */
@import '~react-slidy/lib/index';

/* or directly in your javascript file if you use a bundler */
import 'react-slidy/lib/index.scss'
```

If you're using just css, you could import the CSS directly in your Javascript if you're using a bundler like Webpack:
```js
/* directly in your javascript file if you use a bundler */
import 'react-slidy/lib/styles.css'
```

Or you could load the CSS directly from HTML:
```html
<!-- from the index.html -->
<link rel='stylesheet' href='https://unpkg.com/react-slidy@4.0.10/lib/styles.css'>
```

### Use components as slides

You could use the slider with React components. Just put them inside. Every child will be rendered as slide. For example:

<Playground>
  <ReactSlidy>
    <Number num={1} />
    <Number num={2} />
    <Number num={3} />
    <Number num={4} />
  </ReactSlidy>
</Playground>
